# Stock Management Frontend

## Tech Stack

- Vue 3
- Vuex
- Vue Router
- Axios
- Tabulator
- AG Grid

## Installation

Clone:

```bash
git clone https://github.com/jigishapatel2120/stock-management-frontend.git
```

Move to project:

```bash
cd stock-management-frontend
```

Install packages:

```bash
npm install
```

Run:

```bash
npm run dev
```

Open:

http://localhost:5173
